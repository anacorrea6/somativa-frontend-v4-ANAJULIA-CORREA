 Cinepipoca
Este é um repositorio criado especialmente para prova de front-end 4.

Tecnologias Utilizadas
HTML5 (Semântica e Acessibilidade)
CSS3 (Estilização e Feedback Visual)
JavaScript (Manipulação de DOM e Eventos)

Estrutura 
O projeto foi organizado como:  / ├── index.html └── src/ ├── styles/ (Estilos da página) ├── scripts/ (Lógica de interação) └── images/ (Ativos visuais)

 Funcionalidades Implementadas
 interação com as imagens ao passar o mouse
Avisos ou mensagens de saudação de acordo com o horario
questionario que processa e exibe resultados assim que o usuário digita.
Injeção de Conteúdo: Adição de novos itens na tela utilizando manipulação de DOM.
